soprano.calculate.powder.triavg module¶

triavg.py

Contains a class to define the the POWDER averaging algorithm for NMR spectra, as described in:

D. W. Alderman, Mark S. Solum, and David M. Grant Methods for analyzing spectroscopic line shapes. NMR solid powder patterns [J. Chern. Phys. 84 (7), 1 April 1986]

get_orient_angles(N)¶

Generate and return the TriAvg angles (in the form of angles in radians), weights, and triangles.

Args:
N (int): number of divisions on each side of the octahedron used to
generate angles. Higher numbers lead to greater precision.
Returns:
points, weights, tris (np.ndarray): arrays containing respectively the
direction cosines for each
orientation, the weights, and the
triangles (in form of triplets of
indices of the first array)
